# monorepo

[![License][license-badge]][license-url]

> A playground for Monorepo with Yarn Workspaces.

## Development

- Clone the repo

```bash
$ git clone https://github.com/robertoachar/monorepo.git
```

- Install dependencies

```bash
$ yarn
```

- Run

```bash
$ yarn start
```
